Page Development

Hello again, people. If you got redirected to this page then that is good if not then that is fine as well (kind of). This page is mainly about how I developed this website from a basic Portfolio page.

HTML is the source of this web page and is the most preferably used when making basic or intermediate websites like this one. All you need is a text editor (in my case PyCharm but any will do) and a web browser like Google Chrome (https://google.com) but again any will do (firefox, internet explorer, etc). Before this website I had made a portfolio website. The link to this is: http://localhost:63342/loops%20activity/Portfolio.html/index.html?_ijt=upg00i40v2i8jf8l7qapnbpp0i. If you like this web the source Html and Css code for one of my other portfolio website I made before this one is attached. The HTML: https://repl.it/repls/PlumHonestDos#index.html and the Css: https://repl.it/repls/PlumHonestDos#style.css. Css is wonderful way to splash some color into your website and add cool transitions and also add little animations. (An example is when you touch the pictures on the Home page.)

Whereas HTML was the foundation of this website, the real website is made to look like this because of a powerful tool called CSS. CSS is a designer's app on all text editors which makes your web look stunning. If you install Pycharm and don't find CSS when you got to 'File ', 'New ', do not panic. Do the same thing you would do to open a Python File (which is 'file' then 'new') but instead of pressing 'Python file' press 'File' and enter 'styles.css'. You can call it anything but DON'T FORGET THE '.css'! This can also be applied to javascript but sadly there is no use of that in Portfolio Home or About pages. Just the file name (e.g 'main') then add '.js'.Final outcome (e.g 'main.js'). Easy as that can you believe it?
